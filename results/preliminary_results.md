# Preliminary Simulation Results

This document summarizes the non-confidential experimental structure of the current UAV–UGV swarm coordination study.

The full quantitative results are currently kept private while the manuscript is being prepared for academic submission.

---

## Experiment 1: Communication Reduction

**Goal:**  
Compare event-triggered stigmergic communication against continuous broadcasting.

**Evaluation focus:**

- total number of communication events
- communication reduction percentage
- preservation of useful UGV guidance
- effect of event-trigger thresholds

**Status:** Completed in simulation.

---

## Experiment 2: Task Completion

**Goal:**  
Evaluate whether the shared guidance field improves UGV task completion compared with unguided behavior.

**Evaluation focus:**

- task completion time
- path efficiency
- guidance effectiveness
- comparison against baseline behavior

**Status:** Completed in simulation.

---

## Experiment 3: Scalability

**Goal:**  
Evaluate system behavior as the number of agents increases.

**Evaluation focus:**

- swarm size
- communication growth
- coordination stability
- guidance field usefulness

**Status:** Completed in simulation.

---

## Experiment 4: Robustness Under Agent Failure

**Goal:**  
Evaluate whether the coordination approach remains useful when one or more UAVs fail.

**Evaluation focus:**

- UGV task continuation
- guidance degradation
- communication behavior after failure
- robustness of distributed coordination

**Status:** Completed in simulation.

---

## Summary

The preliminary simulation results suggest that event-triggered stigmergic coordination can reduce communication load compared with continuous broadcasting while still providing useful guidance for UGV navigation.

Detailed numerical results will be added after manuscript submission or public preprint release.
